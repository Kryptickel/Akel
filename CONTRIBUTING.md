# Contributing to AKEL Panic Button

Thank you for your interest in contributing to AKEL! We're **95% complete** and need your help to reach 100% and launch to the public.

## 🎯 Current Focus: The Last 5%

We're in the **polish and launch phase**. The most valuable contributions right now are:

### 🔴 Critical (Urgent - Launch Blockers)
- **Security Testing** - Help us find vulnerabilities before public launch
- **Accessibility Testing** - Test with real screen readers, switch control, etc.
- **Device Testing** - Test on different Android/iOS versions and devices
- **Translation** - Help translate to your language (100+ needed)
- **Documentation** - Improve installation guides, API docs, user manuals

### 🟡 Important (Quality Improvements)
- **Bug Reports** - Find and report bugs (especially edge cases)
- **Performance Testing** - Test app under stress conditions
- **UI/UX Feedback** - Suggest improvements to user experience
- **Code Review** - Review pull requests and provide feedback
- **Unit Tests** - Write tests for untested code

### 🟢 Nice to Have (Future Features)
- **Feature Suggestions** - Ideas for post-launch features
- **Integration Ideas** - Suggest third-party integrations
- **Tutorial Content** - Create video tutorials, blog posts
- **Community Building** - Help grow the AKEL community

## 📋 Contribution Process

### 1. **Before You Start**

- Read our [Code of Conduct](CODE_OF_CONDUCT.md)
- Check [existing issues](https://github.com/Kryptickel/akel/issues) to avoid duplicates
- For major changes, open an issue first to discuss

### 2. **Setting Up Development Environment**

```bash
# Fork the repository on GitHub
# Clone your fork
git clone https://github.com/YOUR-USERNAME/akel.git
cd akel

# Add upstream remote
git remote add upstream https://github.com/Kryptickel/akel.git

# Install dependencies
flutter pub get

# Run the app
flutter run
