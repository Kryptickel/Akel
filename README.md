# Akel - Panic Button Safety App

A comprehensive emergency safety application designed to provide instant help and protection in crisis situations.

## 🚨 Core Mission

Akel is a multi-platform panic button app that provides comprehensive emergency response capabilities, connecting users with emergency services, trusted contacts, and safety features during critical situations.

## ✨ Key Features

### 🔐 User Management & Setup
- **Startup Profile Registration**: Complete user profile setup with personal information and photo
- **Multi-Device Synchronization**: Link and manage multiple devices (phones, tablets, PCs)
- **Cloud Storage Integration**: Secure data backup and synchronization

### 🚨 Emergency Response
- **Panic Button Activation**: Instant emergency trigger with multiple activation methods
- **Silent Mode**: Discreet emergency activation without visible indication
- **Voice Activation**: Hands-free emergency triggering with voice commands
- **Widget Access**: Quick home screen widget for fast activation

### 📱 Device Management
- **Hibernation Mode**: Lock device into emergency mode until remotely deactivated
- **Remote Control**: Manage all linked devices during emergencies
- **App Migration**: Transfer applications between linked devices
- **Device Tracking**: Real-time location monitoring of all connected devices

### 🔊 Audio & Video Monitoring
- **Full Audio/Video Access**: Automatic recording during panic mode
- **Real-time Streaming**: Live camera and microphone feed to emergency contacts
- **Evidence Collection**: Secure storage of incident recordings

### 🗺️ Location & Navigation
- **GPS Tracking**: Continuous location monitoring and sharing
- **Map Integration**: Nearby police stations, hospitals, and emergency services
- **Geo-fencing**: Safe zone alerts and boundary monitoring
- **Emergency Routing**: Optimal escape route planning

### 👥 Emergency Network
- **Emergency Contacts**: Automatic SOS alerts to trusted contacts
- **Multi-User Connections**: Shareable links for emergency contact networks
- **Group Coordination**: Real-time updates and coordination with trusted contacts
- **Emergency Broadcasting**: SOS signals to nearby app users

### 🔒 Security & Privacy
- **Cloned SIM Support**: Backup connectivity options
- **Secure Wi-Fi**: Protected connections and VPN integration
- **Anti-Keylogger**: Protection against malicious key logging
- **Device Integrity**: Tamper detection and security monitoring
- **Decoy Mode**: Fake interface to disguise emergency actions

### 🏥 Medical & Health
- **Medical Profile**: Emergency health information and medical ID
- **Hospital Locator**: Nearby medical facilities with navigation
- **Medication Reminders**: Health management and appointment tracking
- **Crash Detection**: Automatic accident detection and response

### 🌐 Advanced Features
- **AI Threat Detection**: Environmental scanning for potential dangers
- **Scam Protection**: Dark web monitoring and phishing detection
- **App Sandboxing**: Malware and spyware detection
- **Emergency Translation**: Multi-language emergency communication
- **Legal Aid Integration**: Access to legal professionals

### 👶 Specialized Modes
- **Child Safety Mode**: Parent tracking and tamper-proof features
- **Disaster-Specific Modes**: Earthquake, flood, fire response protocols
- **Self-Defense Training**: Safety tutorials and exercises

## 🏗️ Technical Architecture

### Platform Support
- **Mobile**: Android, iOS native applications
- **Desktop**: Windows, macOS, Linux desktop applications  
- **Web**: Progressive Web App (PWA) support
- **Cross-Platform**: Flutter framework for unified codebase

### Backend Infrastructure
- **Cloud Provider**: Firebase/AWS for scalable backend services
- **Real-time Communication**: WebSocket connections for instant alerts
- **Data Storage**: Encrypted cloud storage for user data and media
- **API Integration**: Emergency services and map provider APIs

### Security Features
- **End-to-End Encryption**: All communications and data encrypted
- **Biometric Authentication**: Fingerprint and face ID security
- **Zero-Knowledge Architecture**: Privacy-first data handling
- **Compliance**: GDPR, HIPAA, and emergency service regulations

## 🚀 Installation & Setup

### Prerequisites
- Flutter SDK 3.0+
- Dart 3.0+
- Firebase/AWS account for backend services
- API keys for map services (Google Maps, Apple Maps)

### Development Setup
```bash
# Clone the repository
git clone https://github.com/Kryptickel/Akel.git
cd Akel

# Install dependencies
flutter pub get

# Run on mobile
flutter run

# Run on web
flutter run -d chrome

# Run on desktop
flutter run -d windows/macos/linux
```
